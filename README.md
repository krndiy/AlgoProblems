# AlgoProblems

Repository containing optimized solutions to algorithmic problems from different platforms such as LeetCode, Codewars, Codeforces, etc.

Each platform has its own folder and each solution includes time and space complexity analysis, test case coverage, and implementation notes. 

## Repository Structure
```
AlgoProblems/
├── codeforces/
├── leetcode/
│   ├── easy/
│   ├── medium/
│   └── hard/
└── codewars/
```

## Languages
Primarily used languages are Python 3 and C++17, PL/SQL and PL/pgSQL for database problems.

## Usage
``` bash
git clone https://github.com/krndiy/AlgoProblems.git
cd AlgoProblems/{leetcode,codewars,codeforces}
```
